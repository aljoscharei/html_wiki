install a linux launcher

Install pi-apps wget -qO- https://raw.githubusercontent.com/Botspot/pi-apps/master/install | bash Then in the games submenu find the "GDLauncher" and install it.

The launcher downloads Java 17, all Playser who want to join the world need to have the same version. The maximal version compatible with Java 17 is 1.19.2.
