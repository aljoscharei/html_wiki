Setting up a super-lightweight personal knowledge base

  • Choose a directory
  • create an index.html file (necessary for github pages
  • start crating html files with links
  • you can directly push your files to a new github repo
  • in the settings of the new repo, deploy from branch
  • go to https://aljoscharei.github.io/html_wiki this will show the index.html file
  • if you need to change a filename, you can change all the links with running (on a mac):find . -type f -name "*.html" -exec sh -c 'LANG=C LC_ALL=C sed -i "" "s/second file/third file/g" "$0"' {} \;
